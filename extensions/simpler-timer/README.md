# Simpler Timer

Set timer for your job and use notification to alarm.

## Features

- **Quick Timer Setup**: Easily set timers using natural language inputs (e.g., "45m Take a break", "1h Work").
- **Timer Management**: View all active timers in a list, check remaining time, and cancel them if needed.
- **Native Notifications**: Receive system notifications when your timer finishes.

## Commands

### Set Timer

Set a simple timer for notification.

**Arguments:**
- `input`: Time duration followed by an optional message (e.g., `20m Nap`, `1h30m Focus`).

### Manage Timers

List and manage active timers. Allows you to see the progress of running timers and cancel them.
